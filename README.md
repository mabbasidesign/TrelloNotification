# TrelloNotification Azure Functions

This project contains Azure Functions written in C# for handling Trello-style notification events (order, product, inventory, etc.).

## Getting Started

- Requires .NET 6.0 and Azure Functions Core Tools.
- To run locally: `func start`
- To add new functions: `func new`

## Structure
- SampleNotification: HTTP-triggered sample function

## Documentation
See the official Azure Functions C# docs: https://learn.microsoft.com/azure/azure-functions/functions-dotnet-class-library
